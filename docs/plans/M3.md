# M3 — Opportunity + Pipeline + Lost reason

## Zadaci
- [x] Prisma: Opportunity (FK Stage + LostReason), Stage i LostReason kao FK tabele.
- [x] Stage sekvenca: NEW → QUALIFIED → PROPOSAL_SENT → NEGOTIATION → VERBALLY_CONFIRMED → WON / LOST.
- [x] Weighted value = expValue × probability / 100 (auto).
- [x] Lost reason validacija: obavezan pri prelazu u LOST; „OSTALO" traži slobodan tekst.
- [x] Side-effect na Stage change: WON → HOLD rezervacije u CONFIRMED; LOST → HOLD u RELEASED.
- [x] Konverzija WON → Kampanja sa nasleđenim vrednostima (partner, naziv, valuta).
- [x] Pipeline kanban (kolone po Stage-u, kartice sa partner/vrednost/probability).
- [x] Pipeline funnel agregacija (count + weighted per stage).
- [x] UI: opportunity list + create modal + detail sa setStage + convertToCampaign.
